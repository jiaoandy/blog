分享一些日常开发中实用的 JavaScript 技巧。

## 数组去重

```javascript
const unique = [...new Set(arr)];
```

## 可选链操作符

```javascript
const name = user?.profile?.name ?? "匿名";
```

## 对象解构

```javascript
const { title, date, tags } = post;
```

## async/await 错误处理

```javascript
const [error, data] = await fetchData().then(
  data => [null, data],
  error => [error, null]
);
```
