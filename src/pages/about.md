## 关于本站

这个博客使用 [Eleventy (11ty)](https://www.11ty.dev/) 构建，托管在 [GitHub Pages](https://pages.github.com/) 上。

### 技术栈

- **静态站点生成器** — Eleventy (11ty)
- **模板引擎** — Nunjucks
- **样式** — 纯 CSS，CSS 自定义属性
- **部署** — GitHub Actions → GitHub Pages
- **内容格式** — Markdown + YAML Frontmatter

### 设计原则

1. **配置驱动** — 所有站点设置集中在 JSON 数据文件中，零硬编码
2. **组件复用** — 文章卡片、标签徽章、分页等组件全局复用
3. **模块化扩展** — 自定义模块系统，添加新功能无需修改核心模板
4. **简洁高效** — 零客户端框架，纯静态 HTML，快速加载
